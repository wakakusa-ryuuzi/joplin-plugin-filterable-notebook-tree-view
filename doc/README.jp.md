# Joplin Plugin Filterable Notebook TreeView

> [!CAUTION]
> このプラグインは開発中です。
> データを破壊してしまう様な事は無いと思いますが、自己責任でご使用ください。

Filterable Notebook TreeViewプラグインは、ノートブックのツリーを名前でフィルタ出来るパネルを追加します。

![demo](./image/demo.gif)


# モチベーション

Joplinをヘビーユーズしている事でノートブックが増えすぎて探すのが大変だが

- パネルとして見える
- 親・子を一緒に見たい

と思い、コマンドパレットとは別の絞り込み方法を作りたかったです。


# インストール方法

Joplinのプラグインとして登録しているので、Joplinの「オプション」 -> 「プラグイン」画面で「Filterable Notebook TreeView」と検索してください。


# 使用方法

## パネル

プラグインをインストールして有効化すると、パネルが表示されます。
「表示」 -> 「表示レイアウトをカスタマイズ」から、好みの場所に配置してください。


## フィルタ

「Notebook Name」と書いてある欄に、文字を入力してください。
その文字が含まれるフォルダ＋そのフォルダのルートまでの親が表示されます。

パネル内でフォルダを選択すると、そのフォルダに移動できます。
